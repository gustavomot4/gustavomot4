### 🧭 Sobre mim

- 🎓 Desenvolvedor em formação, transitando de projetos acadêmicos para aplicações completas em produção.
- 🏪 Meu projeto principal, o **SPO — Sistema Pimenta Ousada**, é um sistema de gestão de estoque e vendas usado por um negócio real, com PDV, controle de estoque, relatórios financeiros e autenticação por PIN.
- 🤖 Tenho explorado **"vibe coding"**: dirigir agentes de IA especializados (planejamento → banco de dados → implementação → QA adversarial) para entregar software com verificação em cada etapa, em vez de escrever tudo linha a linha.
- 🧩 Já passei por Java, JavaScript, COBOL e TypeScript — gosto de entender a linguagem certa para cada problema, não só a mais confortável.
- 🌱 Atualmente aprofundando conhecimentos em **Next.js, Prisma e Docker** para levar projetos até produção de forma independente.
